   - SDK: Gradio
   - Hardware: CPU Basic
   - Visibility: Public
3. **Upload Files**: Upload all files from this directory
4. **Set Environment Variable**: Add `OPENAI_API_KEY` in Space settings
5. **Deploy**: Push files and wait for build
